# openKanzlei
Eine Open-Source Kanzleisoftware

## Ausführen
#### Voraussetzungen
- JDK 17+
- Node.JS 24+

### Docker
*kommt bald*

### Manuelle Ausführung
``` bash
git clone https://github.com/evgeny-shevtsov/openKanzlei
```
``` bash
cd openKanzlei
```
*installiere alle Dependencies*
``` bash
npm run setup
```
*openKanzlei starten*
``` bash
npm run start
```
Nach der Startsequenz ist openKanzlei auf `http://localhost:3000` verfügbar.
